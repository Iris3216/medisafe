# 🩺 MediSafe
**Personal Health Management System on Amazon Web Services**

MediSafe คือระบบจัดการข้อมูลสุขภาพส่วนตัว (Personal Health Management System)  
พัฒนาในรูปแบบเว็บแอปพลิเคชัน โดยใช้บริการของ Amazon Web Services (AWS)  
เพื่อให้ผู้ใช้สามารถบันทึก ติดตาม และจัดเก็บข้อมูลสุขภาพของตนเองได้อย่างปลอดภัย

---

## 📖 รายละเอียดโครงการ
โครงการนี้จัดทำขึ้นเป็นส่วนหนึ่งของรายวิชา  
**ICT24467 การพัฒนาซอฟต์แวร์ระบบประมวลผลคลาวด์และความปลอดภัยของข้อมูล**  
มหาวิทยาลัยศรีปทุม ปีการศึกษา 2568

ระบบ MediSafe ช่วยแก้ปัญหาการจัดเก็บข้อมูลสุขภาพที่กระจัดกระจาย  
โดยรวบรวมข้อมูลทั้งหมดไว้ในที่เดียว สามารถเข้าถึงได้ทุกที่ทุกเวลา  
และมีมาตรการด้านความปลอดภัยระดับคลาวด์

---

## 🎯 วัตถุประสงค์ของโครงการ
- พัฒนาระบบจัดการข้อมูลสุขภาพส่วนตัวที่มีความปลอดภัย
- ศึกษาและประยุกต์ใช้บริการของ Amazon Web Services (AWS)
- พัฒนา Backend API ด้วย Node.js และ Express
- พัฒนา Frontend ด้วย React ที่รองรับการใช้งานภาษาไทย
- นำระบบขึ้นใช้งานจริงบน Cloud Environment

---

## ✅ ความสามารถของระบบ
- สมัครสมาชิกและเข้าสู่ระบบด้วยการยืนยันตัวตน
- บันทึกข้อมูลสุขภาพ เช่น ความดันโลหิต ระดับน้ำตาล น้ำหนัก
- จัดการรายการยาที่ใช้อยู่
- อัปโหลดและจัดเก็บเอกสารทางการแพทย์
- แสดงผลข้อมูลสุขภาพย้อนหลังในรูปแบบกราฟ
- จัดการข้อมูลโปรไฟล์ผู้ใช้

## 🔐 ความปลอดภัยของข้อมูล
- ยืนยันตัวตนด้วย Amazon Cognito
- ใช้ JWT Token สำหรับเรียก API
- เข้ารหัสข้อมูลเอกสารบน Amazon S3
- จำกัดการเข้าถึงข้อมูลเฉพาะเจ้าของบัญชี
- บันทึกการใช้งานระบบ (Audit Log)
